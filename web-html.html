合并单元格

    1.合并单元格三部曲

  • 先确定跨行还是跨列合并
  • 找到目标单元格，写上合并方式=合并的单元格数量。比如：< td colspan="2" >< /td >
  • 删除多余的单元格
2.合并语法
合并方式 语法
跨行合并 rowspan="合并单元格的个数"
跨列合并 colspan="合并单元格的个数"

#列表:

  1. 无序列表ul:
    1. < ul>< /ul>中只能嵌套< li>< /li>，直接在< ul>< /ul>标签中输入其他标签或者文字的做法是不被允许的。
    2. < li>与< /li>之间相当于一个容器，可以容纳所有元素。
  2. 有序列表ol
    1. < ol>标签中的type属性值为排序的序列号，不添加type属性时，有序列表默认从数字1开始排序。
    2. 常用的type属性值分别为是1，a，A，i，I
    3. < ol reversed=“reversed”>中的reversed属性能够让有序列表中的序列倒序排列。
    4. < ol start=“3”>中的start属性值为3，有序列表中的第一个序列号将从3开始排列。
